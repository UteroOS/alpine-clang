# Alpine with Clang

## Installed Packages


* alpine-sdk
* bash
* cmake
* clang
* clang-dev
* dpkg
* gcc
* git
* lld
* make
* man
* man-pages
* musl-dev

## Usage

```sh
$ docker container run --rm -it utero/alpine-clang:latest
```
